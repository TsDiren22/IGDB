ERD

Dit is de ERD die ik heb gemaakt bij mijn project. Mijn doel is om een IMDB website te maken, maar dan voor videogames. Mensen zullen dus een account kunnen aanmaken, en games toevoegen aan hun lijst. Je kan ook zelf films toevoegen, wijzigen, en verwijderen. Een game heeft een titel (string), beschrijving (string), datum van uitbreng (string), en een genre (enum van Genre objecten (string)). Ook heeft het een studio (Studio), en consoles (enum van console objecten(string)). De studio is het bedrijf die de game heeft gemaakt. Zij beschikken over een naam (string), adres(string), en een oprichter(string). Ook heeft het een website (URL) en een datum van oprichting (Date) Een console heeft een aantal gebruikers (int) (buiten de app), een platform waarop de games runnen (string), en een datum waarop het is uitgebracht (date). De laatste entiteit die ik heb staan, is de User met een username, password, email, waarvan alles een string is. Ook heeft het een Id (int)

De links die ik hierbij heb gelegd zijn als volgt:
Een user is gelinkt aan een game. Een User kan een lijst van games aanmaken, om als favoriet op te slaan. Een game kan dus nul of meerdere users hebben, en een user kan nul of meerdere games hebben. De game is ook gelinkt aan een studio. Een studio beschikt over een lijst met nul of meerdere games die zij hebben uitgebracht. De game heeft maar één studio. een Game is ook gelinkt aan een console. Hierbij is ervoor gekozen om in game een lijst met één of meerdere consoles te kiezen. Dit komt omdat er geen game uitgebracht zal worden zonder een console om op te spelen. Een console heeft één of meerdere games. Dit komt omdat er geen console uitgebracht zal worden, zonder dat er een game erop gespeeld kan worden.
